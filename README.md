Repo for ARIA_Signal final project.

This project consists in the extraction of 3 features from a dataset of birdsongs in order to classify them into 2 classes.

There are 40 signals in the `annotated_dataset` folder. This folder is the beginning of the project.

The `code` and `notebooks` folders contain several python code which helped build the extraction and classification pipeline

The `final project` folder contains the final extraction and classification pipeline as well as 2 npz files which contain the data to use.
